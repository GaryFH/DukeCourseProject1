Part 1: Data

TASK:

Assignment is to learn what attributes make a movie popular with the goal of creating a predictive algorithm with the response variable being a movies popularity.

Quoted from the Codebook for Movies dataset: The data set is comprised of 651 randomly sampled movies produced and released before 2016. Some of these variables are only there for informational purposes and do not make any sense to include in a statistical analysis.
Note that after Exploratory data analysis and preliminary preprocessing of the dataframe, the number of variables used for statistical analysis was reduced from 32 down to only 11.

Part 2: Research question

As a Data Scientist for a movie studio it is important to provide my employer with evidence to help improve the odds that the movies we produce will be popular. Therefore the Primary Question is: What linear regression model best predicts a movies popularity - note that this data only allows for correlation predictions and should not be considered to imply causality.

Sub questions:

What accuracy can be expected from this model?

Part 4: Modeling

The Popularity variable is determined by audience_score (audience_rating is less accurate) and imdb_rating. A new variable name Popularity was created and used as the Response Variable. The Popularity variable combines audience_score (possible values 0-100) plus 10*imdb_rating (times 10 since possible values for imdb_rating are only 0-10).

d3<-mutate(d2,popularity=(audience_score+(imdb_rating*10)))
d3<-select(d3,popularity,everything())
d4<-select(d3,-audience_score,-imdb_rating)

1) I was able to predict the popularity of a movie based on the final model

2) The analysis shows that the model meets statistical standards and can be considered valid.

3) Note that the Rsquared for the final is only .284 which indicates that only about 28% of the information that explains the response variable is found in this model.
